dotfiles for setups at home, work and beyond.

Setup:
=====

1. Clone the repo
2. Link all bin files
3. Vim setup
  1. Install neovim
  2. Install neovim and neovim-remote from PyPI
4. `exuberant-ctags` is required for unite-outline to support tags for C family
