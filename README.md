dotfiles for setups at home, work and beyond.

Setup:
=====

1. Clone the repo
2. Link all bin files
3. Vim setup
  1. Link vim dir
  2. `git submodule update --init --recursive`
  3. Run `make` inside vim/bundle/vimproc/
  4. To update vim plugins, use `git submodule update --remote` (`gitsbup`
alias)
