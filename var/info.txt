Connecting to any wifi network without wpa_supplicant
1. change /etc/network/interfaces from wpa_supplicant to:
    allow-hotplug wlan0
    iface wlan0 inet dhcp
    wireless-essid any
2. sudo ip link set wlan0 up
3. sudo ifconfig wlan0 up
4. sudo iwlist wlan0 scan
5. sudo iwconfig wlan0 essid "<network name>"
6. sudo dhclient -v wlan0
